# 南网在线 APP — 功能架构与原型

> 仿照南方电网「南网在线」APP 的功能架构设计，包含完整功能架构图、详细功能说明、高保真原型及应用架构文档。

## 📁 项目结构

```
nanwang-online-app/
├── README.md                          # 项目说明
├── docs/
│   ├── architecture.md                # 功能架构图（Mermaid）
│   ├── feature-details.md             # 详细功能清单
│   └── app-architecture.md            # 应用架构文档
├── prototypes/
│   ├── index.html                     # 原型首页（导航）
│   ├── home.html                      # 首页原型
│   ├── billing.html                   # 电费服务原型
│   ├── service.html                   # 业务办理原型
│   ├── repair.html                    # 故障报修原型
│   ├── outage.html                    # 停电信息原型
│   ├── value-added.html               # 增值服务原型
│   └── profile.html                   # 个人中心原型
└── assets/
    └── style.css                      # 全局样式
```
